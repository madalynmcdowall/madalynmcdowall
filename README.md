# Madalyn McDowall

* I’m currently developing computational models for **Binary Black Hole Mergers** in hierarchical triple systems and simulating **Tidal Seismology** in eccentric binaries using `REBOUNDx`.

* My main research focus is on the dynamical origins of **Inclined Hot Jupiters** and the hydrodynamics of protoplanetary disks (**Diskoseismology**).

* I also build the **Stellar Analysis Pipeline**: a Python-based tool for MCMC mode-fitting, automated flare detection, and spectroscopic reduction.

* I am also committed to creating and documenting **Computational Physics Foundations**, including:
    * High-performance solvers for the **Poisson Equation** and numerical integration in **Fortran**.
    * Iterative **Root-Finding** and least-squares fitting algorithms in **C++**.
    * A library of **Observational Astronomy** tools for CCD calibration and transit modeling.
